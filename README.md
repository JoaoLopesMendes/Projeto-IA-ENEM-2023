# Aplicação de Machine Learning na Identificação de Casos de Superação Acadêmica em Contextos de Vulnerabilidade Social Utilizando Microdados do ENEM 2023

## **João Pedro Lopes Mendes RA: 10400793 / Luis Felipe Cunha RA: 10419514**

O Exame Nacional do Ensino Médio (ENEM) consolidou-se como um dos principais mecanismos de acesso ao ensino superior no Brasil, além de representar uma importante fonte de dados educacionais e socioeconômicos.

Este projeto tem como objetivo analisar fatores associados à superação acadêmica em contextos de vulnerabilidade social, utilizando técnicas de Machine Learning aplicadas aos microdados do ENEM 2023.

A proposta do trabalho consiste em identificar estudantes que obtiveram desempenho significativamente acima da média esperada para sua faixa de renda familiar, buscando compreender quais variáveis sociais, familiares e estruturais apresentam maior relação com esse fenômeno.

Para isso, foram utilizados modelos de classificação como:

- **Random Forest**
- **Regressão Logística**

Além disso, o projeto envolve etapas de:

- **Limpeza e tratamento de dados;**
- **Engenharia de atributos (Feature Engineering);**
- **Análise exploratória;**
- **Treinamento e avaliação de modelos;**
- **Interpretação das variáveis mais relevantes;**

Os resultados obtidos demonstram a influência de fatores socioeconômicos no desempenho acadêmico, reforçando discussões relacionadas à desigualdade educacional no Brasil.

Tecnologias Utilizadas:
- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Scikit-learn**
- **Google Colab**
