# Análise Preditiva de Desemepenho Escolar no ENEM 2023 baseada em Fatores Socioeconômicos
## Repositório para o projeto da disciplina de Inteligência Artificial 7ºJ SI - Noite 1º Semestre 2026

**João Pedro Lopes Mendes RA: 10400793 / Luis Felipe Cunha RA: 10419514**

O Exame Nacional do Ensino Médio (ENEM) consolidou-se como o principal mecanismo de acesso ao ensino superior no Brasil, movimentando em média mais de 4 milhões de inscrições anualmente. Entrentanto, o desempenho acadêmico dos participantes não é um fenômeno isolado, sendo profundamente influenciado por fatores socioeconômicos. Desta forma, esse projeto propõe o desenvolvimento de um modelo para prever o desempenho acadêmico dos candidatos do ENEM 2023 baseado em fatores socioeconômicos, além de identificar quais desses fatores funcionam como os principais preditores de rendimento.
